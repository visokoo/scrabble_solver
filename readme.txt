- Tested with ruby 2.4.2 on macOS

If on a Mac:
Install ruby via: https://www.ruby-lang.org/en/documentation/installation/#homebrew

If on Windows:
Install ruby via: https://www.ruby-lang.org/en/documentation/installation/#rubyinstaller

Run gem install minitest -v 5.10.3

To use:
After unzipping, cd to the dir and run `./scrabble_solver <arg>`

To run tests:
score_test.rb
cd to the rspec dir and run `ruby score_test.rb`

scrabble_solver_test.rb
cd to the rspec/ dir and run `ruby scrabble_solver_test.rb <arg>`
